# Insurance Data Analysis - Power BI Dashboard

## Project Overview
This project focuses on analyzing insurance data to understand customer policies, premium amounts, claim amounts, claim status, and customer behavior using Microsoft Power BI.

The dashboard provides interactive insights to help identify trends in insurance policies, claims, and coverage amounts.

## Tools Used
- Microsoft Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)
- Data Visualization

## Dashboard Features

### Key Performance Indicators (KPIs)
- Total Premium Amount: 5.98M
- Total Claim Amount: 16.91M
- Total Coverage Amount: 600.55M

### Analysis Performed

**1. Premium Amount by Policy Type**
- Analyzed premium contribution across different insurance categories:
  - Travel
  - Health
  - Auto
  - Life
  - Home

**2. Claim Status Analysis**
- Visualized claim distribution based on:
  - Rejected
  - Settled
  - Pending

**3. Active vs Inactive Policy Analysis**
- Compared active and inactive customer policies.

**4. Claim Amount Analysis by Age Group**
- Studied claim patterns among:
  - Adults
  - Elders
  - Young Adults

**5. Policy Performance Table**
- Compared pending, rejected, and settled claims across different policy types.

## Dashboard Filters
Interactive filters are included for:
- Policy Number
- Claim Number
- Customer ID
- Gender

## Insights
- Identified policy types generating higher premium values.
- Analyzed claim trends and customer segments.
- Compared claim outcomes to understand insurance performance.
- Studied customer age groups affecting claim amounts.
